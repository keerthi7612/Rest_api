# REST API -- Simple CRUD Application

This project is a simple REST API built using Node.js and
Express.js.
It supports basic CRUD operations (Create, Read, Update, Delete) on
sample book data.

## Features

-   GET-- Fetch all books
-   POST -- Add a new book
-   PUT -- Update a book
-   DELETE-- Remove a book

## Install Dependencies

    npm install

## Run the Server

Development:

    npm run dev

Production:

    npm start

## API Endpoints

  GET- /api/book (Get all books)
  
  POST- /api/book (Create new book)
  
  PUT- /api/book/:id   (Update a book)
  
  DELETE- /api/book/:id   (Delete a book)

## Notes

-   `node_modules` is not included in this project.
-   After cloning or downloading, run **npm install** to restore
    dependencies.
